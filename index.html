Firebase Datenbank verbinden
Für geräteübergreifende Synchronisation brauchst du eine kostenlose Firebase Realtime Database.

In ca. 4 Minuten eingerichtet:
1. console.firebase.google.com → Projekt erstellen
2. Realtime Database → Erstellen wählen
3. Unter Regeln eintragen: {"rules":{".read":true,".write":true}}
4. Datenbank-URL kopieren (Format: https://xxx-rtdb.firebaseio.com)
